About the visualizer:

This visualizer depicts an adjacency matrix as the graph rendered through DOM Elements.

One core feature is the ability to custom make your own map

with walls a target, and source node with ability to erase nodes.

Some Facts About a Breadth First Search

The first path found via a B.F.S. is considered the shortest path!

A B.F.S. traversal cannot find the shortest path on weighted graphs instead use Dijikstra's

The underlying data structure for a B.F.S. is a Queue.

Color and Visual Key:

Beige represents the open spaces

Dark Gray showcases walls

The end point is red

The start point is green

The breadth-first search is purple

The shortest path is denoted by pink

The steps are the numbers on the grid

Time Complexity:

r = number of rows

c = number of columns

Time: O(rc)

Space: O(rc)
